# 项目结果

## 1. 里程碑

### 2025-10-29

完成 legacy 主体框架搭建并通过运行验证。

- 在 GeForce RTX 3060 上训练约 3 小时后，模型以 `81%-11%-8%`（win-draw-lose）战胜 Random Agent
- 这一阶段的重点是把完整规则、MCTS 和训练闭环跑通

### 2025-12-27

完成 v0 主体框架搭建并通过运行验证。

- 使用 C++ / CUDA 重写关键路径
- 自博弈速度相比 Python 功能性框架提升约 `16×`
- 性能瓶颈从规则引擎 / 合法动作生成推向推理与调度链路

### 2026-02-26

v1 训练主线进入强度迭代阶段。

- `vs_random` 达到 **99.80%** 胜率（2,000 局，低模拟次数设置）
- 自博弈 `decisive_games` 从 **1.23% → 81.72%**
- 80 模型锦标赛冠军为 `model_iter_032.pt`
- v1 相对 legacy 单 GPU 基线达到约 **~1,157×** 吞吐提升（positions/s）

## 2. 能力总结

**方法层面**
- 端到端实现：从规则建模到训练闭环
- MCTS 与神经网络协同的 AlphaZero 风格训练
- 从 Python 原型到 C++/CUDA 高性能路径的两轮演进

**工程层面**
- Staged 训练流水线（self-play → train → eval → infer）
- 多 GPU 自博弈与训练调度（4× H20）
- 评估体系（vs-random、vs-previous、锦标赛 Elo/BT）
- 前后端联通的人机对战

## 3. 持续优化方向

- 棋力能否稳定随数据量和算力增长而提升
- 在较小算力预算下，模型能否稳定战胜更强对手
- 通过 bucketed data strategies 应对高和棋率带来的稀疏监督问题
