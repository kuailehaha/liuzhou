# 六洲棋 ProjShare 演示图片

将前端截图或动图放到本目录，并在
`../liuzhou-mcts-projshare.md` 中用对应图片替换虚线占位框。

建议文件名：

- `board-overview.png`：完整棋盘或人机对战界面
- `placement.gif`：落子阶段
- `mark.gif`：形成方或洲后的标记阶段
- `removal.gif`：棋盘填满后的统一移除
- `movement.gif`：走子阶段
- `tactical-example.png`：可选，“只看眼前”会吃亏的示例局面
- `draw-trend.png`：可选，训练轮次与和棋率/胜负局比例

图片建议：

- 四张规则动图使用一致的棋盘缩放、裁剪比例与播放长度。
- 截图优先使用 16:9 或 4:3，避免把重要棋子放在边缘。
- GIF 可在 Marp 预览和 HTML/PDF 中播放；导出为 PPTX 后通常只保留静态帧。
